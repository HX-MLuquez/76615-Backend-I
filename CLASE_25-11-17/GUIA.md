# MONGO DB

Ver version o tipo de mongo

```bash
mongo --version
```

o

```bash
mongosh --version
```

De tener mongo accedemos al cliente

```bash
mongo
```

Y de tener mongosh accedemos al cliente

```bash
mongosh
```

**Importante**: Si el Mongo Compass no se conecta, podemos detener y/o volver a iniciar con:

```bash
net stop MongoDB
```

```bash
net start MongoDB
```
